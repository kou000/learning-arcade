# 英語 アルファベットなぞり練習仕様（AI実装者向け）

このドキュメントは `#/english/alphabet` のアルファベットなぞり練習についてまとめる。  
`docs/` 配下のため、通常ビルドには含まれない。

## 1. 対象画面とルーティング

- 画面: `src/features/english/views/AlphabetPracticePage.tsx`
- ルート: `/#/english/alphabet`
- TOP: `ArcadeHome` の「えいごれんしゅう」から遷移する。

## 2. スコープ

- 画面で選択できる対象は大文字 `A`〜`Z` の1文字練習。
- 小文字 `a`〜`z` のガイド定義と練習データ構造は残すが、調整中のため `SHOW_LETTER_CASE_TOGGLE = false` で切り替えUIは非表示にする。
- 単語練習UIは今後追加する。
- 結果、最後に選んだ文字は保存しない。

## 3. データ構造

- 文字ガイド: `src/features/english/domain/letterGuides.ts`
  - `LetterGuide` はラベル、注記、画数表示位置、正解ストローク点列を持つ。
  - 大文字と小文字は `LETTER_GUIDE_GROUPS` で別グループとして扱う。
- 練習アイテム: `src/features/english/domain/practiceItems.ts`
  - `PracticeItem` は `kind`, `label`, `units` を持つ。
  - 現在は `kind: "letter"` で、1アイテムに1文字ぶんの `unit` を持つ。
  - 大文字/小文字の切り替えは `ALPHABET_PRACTICE_GROUPS` を使う。現在は画面側のフラグで小文字切り替えUIを隠す。
  - 将来の単語練習では `kind: "word"` とし、`CAT` なら `C/A/T` のように複数 `unit` を並べる想定。

## 4. 判定仕様

- 判定ロジック: `src/features/english/domain/traceJudge.ts`
- Canvas 基準サイズは `520 x 520`。
- ユーザー入力は pointer event で取得し、ストロークごとに点列として保持する。
- 入力点は一定間隔で間引き、ストローク終了時に軽く平滑化する。
- 各画ごとに以下を判定する。
  - DTW距離
  - 開始点距離
  - 終了点距離
  - 長さ比
  - 方向一致度
  - ガイドカバー率
- 判定結果は3段階。
  - `great`: 標準しきい値内で、かなり正確に書けている。
  - `good`: 標準より少し緩い条件内で、おおむね書けている。
  - `retry`: ガイドとの差が大きいため、もう一度書く。
- 画面にはDTWなどの詳細数値は表示せず、各画ごとに100点満点の点数を表示する。
- 必要画数分のストロークを書き終えたら自動判定する。
- 画数チェックは常に有効。
- `お手本を描く` ボタンは実装しない。

## 5. 音声読み上げ

- 選択中の文字は `SpeechSynthesis` を使って英語音声で読み上げられる。
- iPad Safari などの自動再生制約を避けるため、読み上げは `よむ` ボタン押下時だけ実行する。
- 端末の既定音声が日本語音声になることを避けるため、利用可能な `en-*` voice からネイティブ寄りの英語音声を優先して選ぶ。
- 小文字もアルファベットの文字名として読み上げるため、読み上げテキストは大文字化する。
- 音声対応状況や選択中の文字は保存しない。

## 6. 判定しきい値

- 判定しきい値は `AlphabetPracticePage.tsx` の `FIXED_TRACE_THRESHOLD = 20` に固定する。
- 判定のゆるさスライダーは表示しない。
- 文字ごとのしきい値は持たない。

## 7. 変更後チェック

- `npm run build`
- `/#` から `/#/english/alphabet` に遷移できること。
- 大文字/小文字切り替えUIが表示されないこと。
- `A`〜`Z` の切替でガイド、入力、判定結果がリセットされること。
- `よむ` で選択中の文字が英語音声で読み上げられること。
- 必要画数分を書いた時点で自動判定されること。
- `消す` が動くこと。
- `判定する` ボタンが存在しないこと。
- `お手本を描く` ボタンが存在しないこと。
- 既存の `#/soroban` と `#/soroban/register` が壊れていないこと。
