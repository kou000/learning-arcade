# 全国珠算学校連盟 検定ルール（画像からの書き起こし）

※ ユーザー提供画像の表をそのまま文章化したものです。

## 乗算 / 除算 / 見取算 共通
- 出題数・時間（段位以外）
  - 乗算: 20題 / 10分
  - 除算: 20題 / 10分
  - 見取算: 10題 / 8分
- 出題数・時間（段位）
  - 乗算: 40題 / 10分
  - 除算: 40題 / 10分
  - 見取算: 20題 / 8分

## 内容・程度（級別）
### 段位
- 乗算: 法・実あわせて11桁（小数3位・名数円未満四捨五入）
- 除算: 法・商あわせて10桁（小数3位・名数円未満四捨五入）
- 見取算: 6桁〜10桁の加算、加減算（1題10口80字）補数計算1題含む

### 1級
- 乗算: 法・実あわせて11桁（小数3位・名数円未満四捨五入）
- 除算: 法・商あわせて10桁（小数3位・名数円未満四捨五入）
- 見取算: 6桁〜10桁の加算、加減算（1題10口80字）補数計算

### 準1級
- 乗算: 法・実あわせて10桁（小数3位・名数円未満四捨五入）
- 除算: 法・商あわせて9桁（小数3位・名数円未満四捨五入）
- 見取算: 5桁〜9桁の加算、加減算（1題10口75字）補数計算

### 2級
- 乗算: 法・実あわせて9桁（小数3位・名数円未満四捨五入）
- 除算: 法・商あわせて8桁（小数3位・名数円未満四捨五入）
- 見取算: 5桁〜8桁の加算、加減算（1題10口70字）補数計算

### 準2級
- 乗算: 法・実あわせて8桁（小数3位・名数円未満四捨五入）
- 除算: 法・商あわせて7桁（小数3位・名数円未満四捨五入）
- 見取算: 4桁〜7桁の加算、加減算（1題10口60字）補数計算

### 3級
- 乗算: 法・実あわせて7桁（小数3位・名数円未満四捨五入）
- 除算: 法・商あわせて6桁（小数3位・名数円未満四捨五入）
- 見取算: 3桁〜6桁の加算、加減算（1題10口50字）

### 準3級
- 乗算: 法・実あわせて7桁（小数3位・名数円以上・端数処理なし）
  - 前半10題の内小数3題 / 後半10題の内小数3題
- 除算: 法・商あわせて6桁（小数3位・名数円以上・端数処理なし）
  - 前半10題の内小数3題 / 後半10題の内小数3題
- 見取算: 3桁〜6桁の加算、加減算
  - 3桁1口、4桁3口、5桁4口、6桁2口（1題10口47字）
  - 前半/後半で減算各2題

### 4級
- 乗算: 法・実あわせて7桁
- 除算: 法・商あわせて6桁
- 見取算: 3桁〜5桁の加算、加減算（1題10口45字）

### 5級
- 乗算: 法・実あわせて6桁
- 除算: 法・商あわせて5桁
- 見取算: 3桁〜4桁の加算、加減算（1題10口35字）

### 6級

### 7級

- 除算: 法・商あわせて4桁
- 見取算: 2桁揃い加算、加減算（1題10口20字）

- 除算: 法・商あわせて3桁
- 見取算: 2桁揃い加算、加減算（1題7口14字）

### 9級

- 除算: 法・商あわせて3桁
- 見取算: 1桁〜2桁の加算、加減算（1題7口10字）

## 暗算（learning-arcade 実装範囲）

このアプリでは、全珠学連の暗算種目を通常種目とは別の種目として扱う。  
現行実装対象は 1級〜8級のみで、段位・準級・9級・10級は未実装。

### 1級
- 乗暗算: 法・実あわせて5桁、30題 / 4分
- 除暗算: 法・商あわせて5桁、30題 / 4分
- 見取暗算: 3桁以上4桁以内の加減算（1題7口、25字、無名数）、15題 / 4分

### 2級
- 乗暗算: 法2桁×実2桁あわせて4桁、30題 / 4分
- 除暗算: 法2桁・商2桁あわせて4桁、30題 / 4分
- 見取暗算: 2桁以上3桁以内の加算（1題7口、18字、無名数）、15題 / 4分

### 3級
- 乗暗算: 法1桁×実3桁あわせて4桁、30題 / 4分
- 除暗算: 法1桁・商3桁あわせて4桁、30題 / 4分
- 見取暗算: 2桁の加算（1題7口、14字、無名数）、15題 / 4分

### 4級
- 乗暗算: 法1桁×実2桁 15題、法1桁×実3桁 15題。計30題 / 4分
- 除暗算: 法1桁・商2桁 15題、法1桁・商3桁 15題。計30題 / 4分
- 見取暗算: 2桁の加算（1題6口、12字、無名数）、15題 / 4分

### 5級
- 乗暗算: 法1桁×実2桁あわせて3桁、30題 / 4分。実には0を使用しない
- 除暗算: 法1桁・商2桁あわせて3桁、30題 / 4分
- 見取暗算: 2桁の加算（1題5口、10字、無名数）、15題 / 4分

### 6級
- 乗暗算: 法1桁×実2桁あわせて3桁、30題 / 4分。交互に実に0を使用
- 除暗算: 法1桁・商2桁あわせて3桁、30題 / 4分。交互に商に0を使用
- 見取暗算: 1桁・2桁の加算（1題4口、5字、無名数）、15題 / 4分

### 7級
- 乗暗算: 法1桁×実2桁あわせて3桁、30題 / 4分。1題おきに実の1位が0
- 除暗算: 法1桁・商2桁あわせて3桁、30題 / 4分。1題おきに商の1位が0
- 見取暗算: 1桁・2桁の加算（1題3口、4字、無名数）、15題 / 4分

### 8級
- 乗暗算: 前半は法1桁×実1桁、後半は法1桁×実2桁。計30題 / 4分
- 除暗算: 前半は法1桁・商1桁、後半は法1桁・商2桁。計30題 / 4分
- 見取暗算: 1桁の加算（1題4口、4字、無名数）、15題 / 4分。繰り上がりは足すと10になる組み合わせ
