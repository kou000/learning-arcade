## 乗算 / 除算 / 見取算 共通
- 出題数・時間（段位以外）
  - 乗算: 20題 / 10分
  - 除算: 20題 / 10分
  - 見取算: 10題 / 8分
- 出題数・時間（段位）
  - 乗算: 40題 / 10分
  - 除算: 40題 / 10分
  - 見取算: 20題 / 8分
